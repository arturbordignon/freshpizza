## Getting Started

Step by step to run this app in your local

### Install packages

For Installing Packages of the Backend, run:

```
cd backend
```

```
yarn install
```

### Run Backend server

```
yarn dev
```
<br>
For Installing Packages of the Frontend Web, run:

```
cd frontend
```

```
npm install
```

### Run Frontend server

```
npm run dev
```
<br>
For Installing Packages of Mobile App, run:

```
cd mobile
```

```
npx expo install
```
or
```
expo install
```

### Run Frontend server

```
npx expo start
```
or

```
expo start
```
